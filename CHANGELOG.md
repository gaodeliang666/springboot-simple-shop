# Changelog

## v0.1.0
- 完成 Spring Boot 项目基础搭建
- 完成用户模块 CRUD
- 完成商品模块 CRUD
- 完成数据库购物车模块
- 完成订单基础功能
- 接入统一返回结果与统一异常处理

## v0.2.0
- 接入 Redis
- 实现 Redis 购物车
- 支持修改 Redis 购物车数量与删除商品
- 为 Redis 购物车增加过期时间续期
- 实现商品详情缓存
- 实现商品列表缓存
- 增加空值缓存，防止缓存穿透
- 商品新增、修改、删除时同步删除缓存

## v0.3.0
- 接入 RabbitMQ
- 创建订单延迟队列、释放队列与交换机
- 解决 RabbitMQ 交换机和队列未自动声明的问题
- 下单成功后发送延迟取消订单消息
- 监听释放队列，自动取消超时未支付订单
- 对已支付订单增加状态校验，避免误取消
- 规范 RabbitMQ 模块日志与状态常量
- 将延迟消息从裸订单 ID 升级为对象消息

## v0.4.0

### Added
- 接入 Swagger / OpenAPI 接口文档
- 支持在线查看和调试用户、商品、购物车、订单等接口

### Fixed
- 修复 Swagger UI 无法渲染接口文档的问题
- 修复 `/v3/api-docs` 返回 500 的问题
- 修复 springdoc 与 Spring Boot 版本不兼容导致的文档生成异常

## v0.5.0

- 支持用户名密码登录
- 登录成功后生成 JWT token

### Added

- 支持校验请求头中的 token
- 支持解析当前登录用户并保存到线程上下文

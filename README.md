# Simple Shop

基于 Spring Boot 的简易电商系统练手项目。

## 技术栈

* Java 17
* Spring Boot 3
* MyBatis
* MySQL
* Redis
* RabbitMQ
* JWT
* Maven

## 项目简介

这是一个面向电商业务场景的后端练手项目，主要用于实践用户鉴权、商品管理、购物车、订单流转、库存扣减、缓存和消息队列等常见后端技术。

## 核心技术点

* 使用 **JWT** 实现用户登录鉴权
* 使用 **MyBatis + MySQL** 实现商品、购物车、订单等核心数据管理
* 使用 **Redis** 实现商品缓存与 Redis 购物车
* 使用 **RabbitMQ** 实现订单超时未支付自动取消
* 使用 **数据库条件扣库存** 实现基础防超卖
* 使用 **订单状态条件更新** 保证支付、取消等状态流转的并发安全
* 使用 **缓存删除策略** 保证库存变化后的缓存一致性
* 使用 **全局异常处理** 统一接口错误返回

## 适用场景

适合作为 Spring Boot 后端练手项目，也可作为校招 / 初级后端简历项目，用于展示：

* Spring Boot 基础开发能力
* MySQL 与 MyBatis 的实际使用
* Redis 与 RabbitMQ 的业务落地
* 对订单、库存、缓存一致性问题的理解
